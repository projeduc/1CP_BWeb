Style externe

Si on veut appliquer le même style sur plusieurs pages web, on peut recopier le même style dans la balise STYLE sur toutes les pages. Mais, lorsqu'on veut modifier une règle, on doit la modifier dans toutes les pages. Une solution est d'utiliser un style externe. On importe un style externe en utilisant la balise LINK dans HEAD. Par exemple, les DIV suivants sont colorés en vert à partir d'un style externe.
